## Follow these commands to Run the Project



### 1. Clone the Repository
```
git clone https://github.com/AnsariMdRizwan/Roxiller-assignment.git
cd your-repo
```
### 2 move to backend.
```
cd backend
npm install
```
### 3 Start the backend:
```
nodenon server.js
```
### 5 move to frontend(React).
```
cd frontend
npm install
```
### 6 Start the forntent.
```
npm run dev
```
